# NYPD Shooting Incident Analysis

This repository contains a reproducible R Markdown analysis of the NYPD Shooting Incident dataset.  
The project examines trends over time, differences across boroughs, victim demographics, and factors associated with fatal outcomes.

## Files in this Repository

- `NYPD_Shooting_Analysis.Rmd`  
  The reproducible R Markdown document containing all code, analysis, and visualizations.

- `NYPD_Shooting_Analysis.pdf`  
  The knitted PDF output generated from the R Markdown file.

## How to Reproduce the Analysis

1. Download or clone this repository.
2. Open `NYPD_Shooting_Analysis.Rmd` in RStudio.
3. Knit the document to PDF.

The document should knit without errors in a clean R session.

## Required R Packages

The following R packages are used in this analysis:
- `tidyverse`
- `lubridate`
- `hms`
- `scales`
- `broom`
